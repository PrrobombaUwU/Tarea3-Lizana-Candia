## Descripcion 
Este codigo crea un grafo dirigido para encontrar el camino más corto para la construcción de edificios, el usuario debe crear un archivo llamado "archivo.txt" con todas las instrucciones disponibles en la guía de Tarea_3__EDD__2026_01, el programa leerá las instrucciones, añadiendo los edificios como nodos y guardando el tiempo en el nodo respectivo, luego leerá cada linea de instrucciones, añadiendo los arcos (prerequisitos), actualizando pesos y quitando arcos, y actualizando la información cada raiz k de consultas.

## Características Técnicas 
* Lenguaje: C++
* Sistema operativo: Linux(Ubuntu 24.04)

## Como funciona el codigo
Antes de ejecutar el código, debe tener el archivo.txt con la información correspondiente.
